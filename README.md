LavalifeProject est un projet d’application inspiré du concept des plateformes de rencontres en ligne réalisé, c'est une application Web.
L’objectif de ce projet est de développer une application permettant aux utilisateurs de créer un profil, d’explorer d’autres profils et d’interagir avec d’autres utilisateurs.

Fonctionnalités:

      Inscription et authentification des utilisateurs
      
      Création et modification de profils
      
      Recherche et consultation d’autres profils
      
      Système d’interaction ou de messagerie (selon l’implémentation)
      
      Organisation claire du code source
      
      Gestion des données utilisateurs

Technologies utilisées:
        C# pour le backend
        Html, CSS pour le frontEnd
        utilisation du .Net Framework pour gérer la connexion à la base de donnée
